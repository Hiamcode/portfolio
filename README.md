# Personal Portfolio / Curriculum

## 💼 About this project
Este proyecto fue creado para presentar mi perfil profesional como desarrollador web de una forma más dinámica e interactiva que un currículum tradicional.

En lugar de solo listar información, este sitio permite explorar mis habilidades, proyectos y crecimiento de una manera más visual y práctica.
